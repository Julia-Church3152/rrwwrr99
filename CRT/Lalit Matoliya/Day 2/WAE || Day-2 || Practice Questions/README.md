**1.** What will be the result of the following program if the inputs are 2 3?

```c
main()
{
  int a,b;
  printf("Enter two numbers:");
  scanf(“%d%d”,a,b);
  printf(“%d+%d=%d”,a,b,a+b);
}
```

(a) 2 + 3 =5\
(b) Syntax Error\
(c) Will generate run time error/core dump\
(d) A+b=5

_Correct Answer:_ **a**
